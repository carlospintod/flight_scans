PHOSPHOR_

The system behind flight_scans: a flight tracker that reads like a terminal you trust. Dark like a shell, monospace for everything that carries data, one phosphor-green accent that only ever means live, cheap, or go — and copy that states the mechanism instead of selling.

Principles

01

Mono is the data voice

Plex Mono carries every label, number, table, and button. Sans appears only when English wraps to two-plus sentences.

02

Green is an accent

Matrix green marks live / cheap / act — the hero price, headings, primary text, link hover. Never a paragraph or a table body.

03

Every color is a status

No decorative color. One green→amber→red ladder expresses freshness, run health, and budget alike.

04

Depth is background steps

Three grounds — page, surface, raised — do the work of shadows. The only shadow is the green glow, and it means "act now."

05

Honesty is a design element

The product states its own limits in fixed places. Disclaimers instruct verification; they never apologize.

06

No theater beyond the terminal

A blinking cursor, one flying jet, nothing else. Charts don't animate; feedback is an inline caption, never a toast.

·Voice

a scan can spend less, never more
Cost is always a guaranteed upper bound — the ≤N notation plus this refrain. Contingency spend is footnoted "only if the primary rail fails".
the tracker hunts the cheapest day
The system is "the tracker" and its verb is "hunt"; the user is "you". Bad news ships with the remedy in the same sentence.
Not a booking site.
Honesty is a genre: the footer, the "Honest limits" section, stale flags, named blind spots. Disclaimers instruct verification, never apologize.

·Do / Don't

✓Reach for a token's meaning — if it isn't live / cheap / go, it isn't matrix.
✓Wrap any wide element in an overflow-x scroller before shipping.
✓End pages and complex widgets with the fine-print closer.
✓State the fact and the remedy in an error.
✗Colour body text or a table body green.
✗Add a spinner, toast, or modal — swap a label, show a caption.
✗Glow anything amber, red, or cyan. Glow means "live".
✗Apologise in an error message.
